III. Click Event

Pour intéragir avec un clique sur la page web, il faut ajouter un event avec la méthode "addEventListener" comme suit:
"const.addEventListener('click' () => { la réponse } );"

Avec "addEventListener" nous avons aussi vu que nous pouvions changer de style, en ajoutant un style lors de l'évènement exemple: btn1.addEventListener("click", () => { response.classList.add('show-response') response.style.background = "green"; });

IV. Mouse Event

Suivi de la souris:
window.addEventListener("mousemove", (e) => { mousemove.style.left = e.pageX + "px"; mousemove.style.top = e.pageY + "px"; });
Presse de la souris:
window.addEventListener("mousedown", () => { mousemove.style.height = '100px'; });
Relache la pression de la souris:
window.addEventListener("mouseup", () => { mousemove.style.width = '100px'; });
Souris qui passe dans la partie selectionnée:
chapitre1.addEventListener("mouseenter", () => { chapitre1.style.background = 'blue'; });
Souris qui sort de la partie selectionnée
chapitre1.addEventListener("mouseout", () => { chapitre1.style.background = 'pink'; });
Souris qui passe sur la partie selectionnée:
chapitre2.addEventListener("mouseover", () => { chapitre2.style.transform = 'rotate(2deg)'; });

V. keyPress

Pour afficher les touches auxquelles on appuie:
document.addEventListener('keypress', (e) => { key.textContent = e.key; });

VI. ScrollEvent

Pour qu'une navbar se cache:
window.addEventListener('scroll', () => { if (window.scrollY > 120){ scroll.style.top = 0; } else { scroll.style.top = "-50px"; } });
